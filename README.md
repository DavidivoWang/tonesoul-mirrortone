# tonesoul-mirrortone

This repository is being built as the MirrorTone collaboration surface for the ToneSoul baseline.

## Current state

- Repository initialized
- Baseline source archive available locally as `tonesoul52-master.zip`
- Import is being performed in batches
- Priority is to preserve the ToneSoul baseline first, then layer MirrorTone collaboration work on top

## Immediate goal

Turn this repository into a readable, runnable, reviewable working surface based on the `tonesoul52-master` baseline, without silently rewriting ToneSoul's worldview.

## Import policy

1. Preserve the baseline before redesign.
2. Do not start by rewriting philosophy or public narrative.
3. Fix entry-path and onboarding surface first.
4. Keep later collaboration changes auditable.

## First import track

- Add import status and instructions
- Add local zip import helper
- Add initial repository hygiene files
- Then begin importing baseline canonical files in batches

## Notes

Because the current GitHub write connector supports text-file mutations but not direct local binary zip upload, the baseline import is being staged in an auditable way instead of pretending the zip has already been committed.
